健康ダッシュボード（GAS）セットアップ手順
========================================

1. スプレッドシートを新規作成する（学習レポート用と分けるとメニューがぶつからない）。

2. 1行目に次の列名をそのまま入れる:
   日付 | 睡眠時間 | 体調 | 運動 | メモ

3. 拡張機能 → Apps Script でプロジェクトを開き、このフォルダ内の .gs をすべてコピーして貼り付ける。

4. スクリプトの「プロジェクトの設定」でタイムゾーンを「(GMT+09:00) 東京」に合わせる（トリガーも Asia/Tokyo を指定済み）。

5. スクリプトプロパティ（プロジェクトの設定 → スクリプトのプロパティ）に追加:
   - LINE_CHANNEL_TOKEN … LINE のチャネルアクセストークン
   - LINE_TO_ID … 送信先（ユーザーID または グループID）
   任意:
   - SHEET_NAME … ログを書くシート名（空ならアクティブシート）
   - USE_LINE_PLAIN_TEXT … true なら Flex ではなくテキスト送信

6. スプレッドシートに戻り、再読み込み後メニュー「健康ダッシュボード」から
   「昨日の行で送信」で動作確認。

7. 平日朝の自動送信: メニュー「平日9時トリガー（東京）」を1回実行。
   やり直すときは同じメニューを押す（古い同名トリガーは削除してから再作成）。

8. LINE Messaging API の送信先でブロックされていないか、トークンが正しいかを確認。
