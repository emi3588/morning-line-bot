学習レポート自動化（GAS）
========================

■ 毎朝7時に届ける（重要）
  1. 学習記録のスプレッドシートを開く
  2. メニュー「学習レポート」→「毎朝7時トリガーを設定（東京）」を1回クリック
  3. 初回は「承認」や「このアプリを信頼」などの案内が出たら進める
  4. これで毎日 7:00〜8:00 のあいだに sendDailyLearningReport が自動実行される
     （Googleの仕様で「ちょうど7:00ピッタリ」ではなく、この時間帯に1回）
  5. 止めたいときは「毎朝7時トリガーを削除」

■ シート1行目
  日付 / 連続 / 完了コース数 / 完了レッスン / 週間 / 月間 / 累計
  / 昨日の学習 / 理解できたこと / まだ曖昧なこと
  / 「今日のフォーカス」または「今日やる１つ」/ 今日の用語

■ スクリプトプロパティ（必須）
  LINE_CHANNEL_TOKEN
  LINE_TO_ID

■ 任意
  SHEET_NAME
  USE_LINE_PLAIN_TEXT = true … テキストのみ送信（Flexやめる）
  OPENAI_API_KEY … 「AI推敲→送信」メニュー用

■ LINE の見た目
  既定は Flex（シャープなオレンジ系のカード）。HTMLモックと同じではない。

■ ファイル（Apps Script に貼るのは .gs のみ）
  Config.gs / Sheet.gs / FormatReport.gs / FlexFormat.gs / LinePush.gs / Main.gs / OpenAI.gs
